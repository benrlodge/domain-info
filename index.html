Domain-info.js

What we have here is a JavaScript module that will help you get information on the current domain.

Get the code on github.

Methods:

URL: 

 Domain.url();

Protocol: 

 Domain.protocol();

Subdomain: 

 Domain.subdomain();

Host Name: 

 Domain.hostName();

Host: 

 Domain.host(); // includes port number if specified e.g. example.org:8888

Top Level Domain: 

 Domain.topLevelDomain();

Pathname: 

 Domain.pathName();

Contains: true

 var contains = Domain.contains('github'); // Search for a string within the domain url

Matches: true

 var match = Domain.matches('http://benrlodge.github.io/domain-info/'); 
// Checks for an exact match with the current url
