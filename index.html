Domain-info.js

What we have here is a JavaScript module that will help you get information on the current domain.

Get the code on github.

Methods:

URL: 

 var url = Domain.url();

Protocol: 

 var url = Domain.protocol();

Subdomain: 

 var url = Domain.subdomain();

Host Name: 

 var url = Domain.hostName();

Host: 

 var url = Domain.host(); // includes port number if specified e.g. example.org:8888

Pathname: 

 var url = Domain.pathName();
